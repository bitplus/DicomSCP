DICOM 查看器
配置说明

AeTitle
DICOM 应用实体标题，1-16个字符，只能包含字母、数字、下划线和横线。例如：STORESCP
StoreSCPPort
存储服务端口号，范围1-65535。建议使用11112等标准端口
StoragePath
DICOM文件存储路径，必须是有效的目录路径。例如：D:\dicom\storage
TempPath
临时文件存储路径，用于接收文件时的临时存储。例如：D:\dicom\temp

ValidateCallingAE
是否验证调用方AE Title，启用后只允许指定的AE Title连接。默认：false
AllowedCallingAEs
允许的调用方AE Title列表，当ValidateCallingAE为true时生效。例如：["WORKSTATION1", "PACS1"]
ConcurrentStoreLimit
并发存储限制，0表示不限制。建议值：根据服务器性能设置，如4或8
EnableCompression
是否启用压缩，影响存储和传输性能。默认：false
PreferredTransferSyntax
首选传输语法，当接收到压缩格式的图像时，可以选择转换为以下格式存储：
  • ImplicitVRLittleEndian - 隐式小端
  • ExplicitVRLittleEndian - 显式小端
  • JPEGProcess1 - JPEG基线（有损）
  • JPEGProcess2_4 - JPEG扩展（有损）
  • JPEGProcess14 - JPEG无损
  • JPEGProcess14SV1 - JPEG无损SV1
  • JPEGLSLossless - JPEG-LS无损
  • JPEGLSNearLossless - JPEG-LS近无损
  • JPEG2000Lossless - JPEG2000无损
  • JPEG2000Lossy - JPEG2000有损
  • RLELossless - RLE无损
  • DeflatedExplicitVRLittleEndian - Deflate压缩

AeTitle
Worklist服务的AE Title。例如：WORKLISTSCP
Port
Worklist服务端口号，范围1-65535。建议：11113
ValidateCallingAE
是否验证调用方AE Title。默认：false
AllowedCallingAEs
允许的调用方AE Title列表。例如：["MODALITY1", "MODALITY2"]

AeTitle
查询检索服务的AE Title。例如：QRSCP
Port
查询检索服务端口号。建议：11114
ValidateCallingAE
是否验证调用方AE Title。默认：false
AllowedCallingAEs
允许的调用方AE Title列表
MoveDestinations
C-MOVE目标节点配置，包含：
  • Name - 节点名称
  • AeTitle - 节点AE Title
  • HostName - 主机名或IP
  • Port - 端口号
  • IsDefault - 是否默认节点

LocalAeTitle
本地AE Title，用于 QueryRetrieveSCU 和 CStoreSCU。 例如：QUERYSCU
LocalPort
本地端口号，用于 QueryRetrieveSCU 和 CStoreSCU。 建议：11115
RemoteNodes
远程PACS节点配置，包含：
  • Name - 节点名称，用于显示
  • AeTitle - 远程节点AE Title
  • HostName - 远程主机名或IP
  • Port - 远程端口号
  • IsDefault - 是否默认节点

LogPath
日志根目录。例如：logs
RetainedDays
日志保留天数。默认：31
OutputTemplate
日志输出模板。默认：[{Timestamp:yyyy-MM-dd HH:mm:ss.fff} {Level:u3}] {Message:lj}{NewLine}{Exception}
Services
各服务的日志配置，包含：
  • Enabled - 是否启用
  • MinimumLevel - 最低日志级别
  • EnableConsoleLog - 是否输出到控制台
  • EnableFileLog - 是否输出到文件
  • LogPath - 日志文件路径
  • OutputTemplate - 输出模板

Swagger
API文档配置：
  • Enabled - 是否启用
  • Title - 文档标题
  • Version - 版本号
  • Description - 描述
ConnectionStrings
数据库连接字符串配置：
  • DicomDb - SQLite数据库文件路径
Kestrel
Web服务器配置：
  • Endpoints.Http.Url - 监听地址和端口
